##8.8 strchr/strstr
搜索字符串

	#include <string.h>
	char *strchr(const char *s, int c);
	char *strrchr(const char *s, int c);
	返回值：如果找到字符c，返回字符串s中指向字符c的指针，如果找不到就返回NULL

	#include <string.h>
	char *strstr(const char *haystack, const char *needle);
	返回值：如果找到字串，返回值指向字串的开头，如果找不到就返回NULL
